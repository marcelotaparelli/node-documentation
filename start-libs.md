## Start Libs

**nodemon**
- Atualiza projeto sem precisar rebuildar.

<a href="https://www.npmjs.com/package/nodemon">npm install -g nodemon</a>

- no package.json:
```
  "scripts": {
    "dev": "nodemon yourFile.js",
  }
```

- no terminal:
`npm run dev`
